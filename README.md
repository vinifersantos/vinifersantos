## 🛸 Sobre mim

Sou um desenvolvedor em evolução, com mentalidade comercial, visão de produto e obsessão por construir sistemas úteis de verdade.

Minha pegada é unir **programação, vendas, estratégia, automação e design de experiência** para criar aplicações que não fiquem só bonitas no protótipo, mas que possam operar no mundo real.

Atualmente foco em:

`Sistemas SaaS` · `Apps com auth + banco de dados` · `Dashboards comerciais` · `Produtividade & gestão de tempo` · `Ferramentas internas de operação e vendas` · `Migração de no-code para estruturas sólidas` · `Produtos digitais & automações`

> _“O código é só a nave. O destino é resolver problemas reais.”_

## 🧠 Princípios de desenvolvimento

- Clareza antes de complexidade.
- Sistema bonito é bom. Sistema útil é obrigatório.
- Protótipo sem caminho para produção é castelo de areia.
- Banco de dados, autenticação e deploy precisam ser pensados desde cedo.
- A interface deve parecer simples, mesmo quando o sistema é complexo.
- Código limpo é respeito pelo Vinicius do futuro.
